AutoGen 源码分享讲解稿

面向分享的讲法：先说明版本状态，再讲 Core、AgentChat、Extensions、Studio 四层，最后对比 LangChain、LangGraph 和 CrewAI。

开场定位

AutoGen 是微软开源的多 Agent 框架，当前已经进入 maintenance mode。源码学习价值在于：它把多 Agent 系统拆成 Core Runtime、AgentChat、Extensions 和 Studio 几层，是很典型的“消息运行时 + Agent Team”设计。

目录怎么讲

目录分享口径
autogen-core底层消息运行时，包含 AgentRuntime、RoutedAgent、模型和工具抽象。
autogen-agentchat用户常用 API，包含 AssistantAgent、Team、GroupChat、终止条件。
autogen-ext模型、MCP、代码执行、GraphRAG 等生态适配。
autogen-studio可视化构建和运行多 Agent 应用。

主流程怎么讲

Core Runtime:
AgentRuntime -> send/publish -> queue -> RoutedAgent -> handler

AgentChat:
BaseChatAgent.run -> AssistantAgent -> model/tool loop -> TaskResult

GroupChat:
Team.run_stream -> register participants -> manager selects speaker -> termination
AutoGen 不是只看 AssistantAgent。真正的主线是：Core 负责消息运行时，AgentChat 把运行时包装成 Agent 和 Team，Extensions 再接模型与工具。
